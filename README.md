icytracks
=========

Reads the '.xls' files generated by the Icy 'Export Tracks to XLS' Track Processor and convert it to a data.frame.

Example usage:

``` r
library(icytracks)
data <- read.icy.track.xls('~/icytracks.xls')
head(data, 10)
```
